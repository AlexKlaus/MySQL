<h1>Курсовой проект по MySQL.</h1>
<h2>Требования к курсовому проекту</h2>
<ol>
  <li>Составить общее текстовое описание БД и решаемых ею задач;</li>
  <li>Минимальное количество таблиц - 10;</li>
  <li>Скрипты создания структуры БД (с первичными ключами, индексами, внешними ключами);</li>
  <li>Создать ERDiagram для БД;</li>
  <li>Скрипты наполнения БД данными;</li>
  <li>Скрипты характерных выборок (включающие группировки, JOIN'ы, вложенные таблицы);</li>
  <li>Представления (минимум 2);</li>
  <li>Хранимые процедуры / триггеры;</li>
</ol>

<h2>Описание проекта.</h2>
<p>БД для компании по производству металлорежущего инструмента. 
Данная БД будет отвечать за хранение информации о поступающих заказах, статусе их выполнения, хранить информацию о выпускаемой продукции, 
так же в БД будет вестись учет материала для изготовления на складе и расходных материалов.</p>

<h2>Описание таблиц.</h2>

<h3>customers</h3>
<p>Таблица будет хранить наименования компаний заказчиков.</p>

<h3>carbide_types</h3>
<p>Наименования марок твердого сплава.</p>

<h3>carbide_in_stock</h3>
<p>Информация о материале на складе.</p>

<h3>coating_types</h3>
<p>Виды покрытия изделий.</p>

<h3>product_types</h3>
<p>Виды изготавливаемых изделий.</p>

<h3>media</h3>
<p>Медиафайлы прилагаемые к заказам, чертежи изделий.</p>

<h3>products</h3>
<p>Перечень изделий с подробной информацией.</p>

<h3>orders</h3>
<p>Таблица с заказами(Заказ может состоять из нескольких позиций).</p>

<h3>order_items</h3>
<p>Позиции заказов с описанием изделия, количества и подсчетом необходимого материала.</p>

<h3>carbide_log</h3>
<p>Лог добавлния и рахода материала со склада.</p>

<h2>Триггеры.</h2>

<h3>update_carbide</h3>
<p>Занесение в лог информации об изменении количетва прутков на складе.</p>

<h3>insert_carbide</h3>
<p>Занесение в лог информации о добалении новой записи о сплаве.</p>

<h3>delete_carbide</h3>
<p>Занесение в лог информации об удалении записи о позиции спава.</p>

<h3>insert_order_item</h3>
<p>Вычисление суммарной длины прутка необходимого для изготовления позиции заказа.</p>
